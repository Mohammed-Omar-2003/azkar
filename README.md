# Islamic Web App 🕌

A modern Islamic web application built with React that helps Muslims with their daily عبادات through a clean and interactive interface.

## 📌 Overview

This project is a frontend application that provides essential Islamic tools, including Azkar, prayer times, Qibla direction, and an electronic Tasbeeh counter. It is designed to be simple, fast, and user-friendly.

## ✨ Features

### 📿 Azkar Section

* Morning and Evening Azkar
* Additional Azkar (sleep, after prayer, etc.)
* Organized and easy to read

### 🔢 Electronic Tasbeeh

* Digital counter for Tasbeeh
* Reset and count tracking

### 🧭 Qibla Direction

* Detects user's direction
* Helps find the Qibla easily

### 🕰️ Prayer Times

* Displays daily prayer times
* Based on user location
